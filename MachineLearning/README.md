# where2meet Machine Learning
`Restaurant_Mood.ipynb` is used for model building, while `main.py` is for deployment on Cloud Functions

## Running
For local environment, it's recommended to use [Virtual Environments](https://realpython.com/python-virtual-environments-a-primer)

Run this command to install every packages on `requirements.txt` into pip
```python
pip install -r requirements.txt
```
